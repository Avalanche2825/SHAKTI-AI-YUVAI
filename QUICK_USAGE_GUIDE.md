## ‚ùì TROUBLESHOOTING

### **HELP detection not working:**

- Check microphone permission
- Increase sensitivity (777= ‚Üí Settings)
- Speak clearly and louder
- Check green dot is on

### **Can't view evidence:**

- Check if incident was created
- Try Dashboard ‚Üí Incident Reports first
- Evidence takes ~5 seconds to save

### **Panic button not working:**

- **LONG-PRESS** (hold for 2 seconds)
- Don't just tap it
- Look for vibration feedback

### **Stop feature not working:**

- **LONG-PRESS .** (hold for 2 seconds)
- Try 000= secret code
- Confirm "YES - I'M SAFE"
